
# Places

the Backend side of the places application


## Tech Stack

**Server:** Node, Express


## Run Locally

Clone the project

```bash
  git clone https://github.com/3neef/places-Backend.git
```

Go to the project directory

```bash
  cd places-Backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
